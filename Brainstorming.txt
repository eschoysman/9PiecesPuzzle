===== TEMPLATE - Puzzle =====
=============================

url dedicato? (tipo localhost:3000/challenge, ma in tal caso anche un localhost:3000/explore, entrambi con un header/menu in comune per cambiare tra i due)



Gestione grafica del gioco + comandi di rotation+flip

Due modalità:
	Free Game: solo mettere i pezzi (prima fase)
	Challenge Game: cercare di ottenere una certa soluzione (simile a sopra, ma con un controllo extra)

Indizi (solo per "Challenge Game"):
	- isValid: quello che ho messo (esattamente così com'è): mi porta a una soluzione? (anche per "Free Game")
    - dammi una casella colorata (casuale) / dammi una casella del color specifico / dammi il colore di una casella specifica*
    - dammi l'orientazione di un pezzo (casuale) / pezzo specifico
    - dammi la posizione (e orientazione) di un pezzo (casuale)* / pezzo specifico*

*: sbianca la griglia

API per "Game mode":
	- isValid(solution): in input una soluzione (versione estesa? compatta?) -> return boolean oppure lista delle soluzioni fattibili senza cambiare nulla?
	- cell(solution,color?,position?): restituisce color+position casuali sulla base delle soluzioni fattibili sulla base della grid attuale
	- shape(solution,shape?): restituisce shape+orientazione casuale sulla base delle soluzioni fattibili sulla base della grid attuale
	- piece(solution,shape?): restituisce shape+orientazione+posizione casuale sulla base delle soluzioni fattibili sulla base della grid attuale

Oggetto di tipo "Indizio":
	- soluzione corrente (avrebbe senso?)
	- casella?
	- coloreCasella?
	- shape?
	- shapeOrientation?
	- shapePosition?


Comandi input:
	- Drag&Drop: spostare un pezzo (+controllo se ci sta nella griglia)
	- right clic: rimuovi pezzo dalla griglia
	- R: rotate piece (idealemnte solo quello durante Drag&Drop, come test anche tutti quelli non messi)
	- F: flip piace (idem a R)




Orientazioni/flip:
rotate (90° CW)		-> current = current%4==3 ? current-3 : current+1;
rotate (90° CCW)	-> current = current%4==0 ? current+3 : current-1;
flip				-> current = (current+4)%8
